# ALE2CDL
从AvidLogExchange（ALE）文件格式中提取ASC CDL等级并将其写入ASC-CDL。遵守Pomfort规范

---

中文 | [English](./README_en.md) 


## 如何使用
#### 方法一
```console
python3 ale2cdl.py -i input.ale -o output_directory
```
`-s`  指定`ALE`输入文件路径  
`-o`  指定`output_directory`路径
